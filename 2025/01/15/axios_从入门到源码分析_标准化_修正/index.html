什么是 Axios？

Axios 本质上是一个基于 Promise 的 HTTP 客户端，广泛用于浏览器和 Node.js 环境。它主要用于向服务器发送 HTTP 请求并接收响应。

主要特点：

  • 请求方法支持：

    • GET：用于获取数据。
    • POST：用于提交数据。
    • PUT：用于更新数据。
    • DELETE：用于删除数据。

  • 主要功能：

    • 自动将响应数据解析为 JSON 格式（如果响应是 JSON）。
    • 提供请求和响应的拦截器功能。
    • 支持请求取消。
    • 支持并发请求处理。

  • 它支持多种 HTTP 请求方法，并且适用于浏览器与 Node.js 环境。
