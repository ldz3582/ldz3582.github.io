2. 兄弟组件通信

兄弟组件之间的通信不能直接进行，需要通过 状态管理工具（如 Pinia、Vuex） 或 事件总线。

2.1 使用 Pinia（推荐方式）

插槽是 Vue 中的一种机制，用于实现父组件向子组件传递 HTML 结构或动态内容。
