Author: clock Date: January 28, 2025  12:00:00 Category: 前端

Node.js

虽然不是前端必须掌握的工具，但是为前端开发提供了很多便利,简单了解一下

为什么需要node.js

​ • 前端构建工具： 许多现代前端工具（如 Webpack、Babel、ESLint 等）都基于 Node.js 来运行。你需要理解如何配置这些工具以及如何使用它们来构建和优化项目。

​ • 前后端统一： 如果你学习 Node.js，你就能更好地理解全栈开发。尤其是与前端框架如 React 或 Vue 搭配使用时，了解 Node.js 可以帮助你更好地理解服务器端渲染、API 构建等功能。

Node.js 的核心用途：

​ • 管理项目依赖（npm）。

​ • 运行和配置构建工具（如 Webpack、Babel、ESLint）。

​ • 开发和运行前端相关的命令行工具。

什么是node.js

Node.js

是一个跨平台JavaScript 运行环境，使开发者可以搭建服务器端的JavaScript应用程序

作用：使用Node.js编写服务器端的程序

  • 编写数据接口，提供网页资源浏览功能等等
  • 前端工程化：为后续学习vue和React做铺垫

Node.js模块化

Node.js 使用了 CommonJS 模块规范，这是它的核心模块系统。它提供了模块化的机制，通过 require() 来引入模块，使用 module.exports 或 exports 来导出模块。
